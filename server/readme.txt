--------------
INSTALL DATABASE
--------------

Download the latest production release of MongoDB from :
https://www.mongodb.com/download-center?_ga=2.123194891.1822248697.1522395660-2086062422.1522395660#production

Open the file to launch the wizard.

MongoDB requires a data directory to store all data.
Create the directory "/data/db" on the drive from which you start MongoDB.

--------------
START APPLICATION
--------------

For start the application start a local instance of MongoDB, open a terminal in this folder and type:

npm install
npm start


Go to http://localhost:3000


--------------
CONFIGURE
--------------

Set database config in

    properties.js





--------------
START WITH DOCKER
--------------

To start the application with Docker:

 - Install Docker

 - Run command:
    docker-compose up

 - Go to 
    http://localhost:80


    After code changes you need to rebuild container running:
        docker-compose up --build
